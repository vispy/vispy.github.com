Third Party Projects#

External projects that extend VisPy’s capabilities are listed below.

If you would like to have your project added to this list, please let us know by either making an issue on GitHub or, even better, making a pull request with the addition to this page.

Tools#

These are standalone applications that utilize VisPy.

DIVE#

DIVE is an extendable Qt GUI designed to simplify the process for plotting pandas DataFrames in VisPy.
Key features include: 2D/3D axes, animated plots, and data filtering/selection.
