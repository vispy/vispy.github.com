    Community¶

    If you have questions related to the use of vispy, we have a chat room on gitter.

    If you find a problem related to this website, please submit an issue to the website repository.

    If you have a bug to report, a feature you’d like to request or a patch you wish to submit, please submit an issue
    If you want to help with the development of vispy, join us on GitHub. There is also a developer's mailing list.
